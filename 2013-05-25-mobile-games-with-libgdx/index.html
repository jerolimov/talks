Heads Up

libGDX is a crossplattform game development framework.

Box2D is a 2D physics engine for games.

## Downloads 1. Install the Java JDK with your package manager or [download it](http://www.oracle.com/technetwork/java/javase/downloads/index.html). 2. You need a Java IDE, for example the [Android SDK](http://developer.android.com/sdk/index.html) * which includes the Android Developer Tools * and a preconfigured [Eclipse SDK](http://www.eclipse.org/downloads/) 3. [libGDX](http://libgdx.badlogicgames.com/download.html) (includes compatible Box2D) 4. Box2D [Physics Body Editor](https://code.google.com/p/box2d-editor/) 5. Our [demo sprites](https://www.dropbox.com/sh/0xgnvbgo9ezmowm/3L3qPDbYjk) -- All available here: http://bit.ly/meteors-downloads Sourcecode: http://bit.ly/meteors-code
## Prepare the Android platform * Extract the android SDK * Start eclipse and open the Android SDK manager * Select Android 4.0.3 (API 15) -> SDK Platform, optional more * Install the package(s)
## Prepare libGDX * Extract the libgdx and physics-body-editor :-) * Test if libgdx/gdx-setup-ui.jar works * Test if physics-body-editor/physics-body-editor.jar works

    • Android and dependent systems like Google TV and OUYA.
    • iOS
    • Web
