Eine Übung zu den Entwicklungstools

Ein kleiner Satz an Aufgaben um euch zu demonstrieren, wie ihr mit den Entwicklungstools des Google Chrome Browsers ein HTML Dokument sichten, bearbeiten und auf Fehler überprüfen könnt.

Aufgabe 1 - Farben Wirrwar

Die unteren Texte sollen in den Farben der MI dargestellt werden, leider hat sich anscheinend irgendwo im HTML Code ein Fehler eingeschlichen? Schaut euch im Firebug/Developer Tools (Chrome) den Code an und ersetzt die falschen CSS Klassen mit den richtigen.

Aufgabe 2 - Wo stehe ich?

Hier fehlen die Klassenbezeichnungen! Schaut euch in der CSS mögliche Klassen an die in Frage kommen und ersetzt das "?" durch den entsprechenden Klassennamen für das "p" Tag

Ich mag die linke Seite des Bildschirms wirklich gerne :)

Ich mag die rechte Seite des Bildschirms wirklich gerne :)

Aufgabe 3 - Bilderrätsel

Beim Positionieren des 2. Bildes hat sich ein Fehler eingeschlichen! Nutzt die "padding" Angaben um das Bild anhand des 1. Bildes auszurichten.
